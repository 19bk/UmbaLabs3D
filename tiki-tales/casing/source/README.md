# Source files

Baseline files imported by every release in this folder. Treat as read-only — modifications belong in a new versioned folder.

| File | Purpose |
|---|---|
| `original-lid.stl` | Unmodified lid from 2.0.0 baseline |
| `original-base.stl` | Unmodified base from 2.0.0 baseline |
| `base.scad` | v5 speaker base parametric OpenSCAD source |
| `bunny-lid-variant.stl` | Alternative lid shape (not on release track) |
| `fig-cup-variant.stl` | Alternative figurine cup (not on release track) |

All release `lid.scad` files import `../source/original-lid.stl`.
