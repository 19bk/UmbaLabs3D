FAQs

Quick answers before you start.
What file types do you accept?

Most common formats work: STL, OBJ, and STEP. If you have something else, send it anyway and we will advise.

How long does a print take?

Turnaround depends on size and material. Small items can be same-day or next-day, while large builds take longer.

Can you help with design?

Yes. Send a sketch, reference photo, or concept and we can build the 3D model for you.

Is my design confidential?

We handle files privately and only use them for production unless you request a showcase.
