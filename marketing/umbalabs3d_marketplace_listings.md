# UmbaLabs 3D - Marketplace Listings

Purpose
- Standardize listings for Jiji, Jumia, and Kilimall.
- Focus on repeatable products and clear fulfillment rules.

Global Listing Rules (apply to all platforms)
- Use real photos only (no contact numbers or watermarks on images).
- One product per listing.
- Clear price range and size options.
- Lead time and delivery note in every listing.

Core SKU Catalog (repeatable items)
1) Desk cable organizer (3-slot)
2) Phone stand (adjustable)
3) Earbuds case holder
4) Keychain with name
5) Desk nameplate
6) Plant pot (small, geometric)
7) Pen holder
8) Card holder
9) Wall hook (2-pack)
10) Drawer organizer tray
11) Mini display stand (for products)
12) Custom logo keychain (business)

Photo Shot List (per SKU)
- Front product shot on clean background
- Angled shot showing detail
- Scale shot with a common object (coin or pen)
- Usage context shot (desk or shelf)

Pricing Model (starter)
- Base price + material cost + print time cost + finishing fee
- Use a minimum order value and add a rush fee for 24-48 hour delivery
