# UmbaLabs 3D - Concrete Build Plan

Goal
- Build a single-page site that converts to WhatsApp quotes, launch repeatable SKUs on marketplaces, and create a B2B outreach kit.

Assumptions
- Primary market: Nairobi, Kenya.
- Primary CTA: WhatsApp.
- One production printer with a weekly capacity limit.

Phase 1 - Week 1 (Foundation)
1) Finalize offer and positioning
   - Choose 3 primary services: Custom prints, Prototypes, Architectural models.
   - Confirm turnaround policy: standard and rush.

2) Brand and messaging
   - Use existing colors and fonts from brand kit.
   - Pick a single hero headline and short subhead.

3) Website
   - Implement a single HTML landing page.
   - Add placeholders for real WhatsApp number and testimonials.

Deliverables
- files/umbalabs3d_website.html

Phase 2 - Week 2 (Marketplaces)
1) SKU selection
   - Choose 8-12 repeatable products with low failure risk.

2) Listing templates
   - Create listing templates for Jiji, Jumia, Kilimall.
   - Add photo shot list and sizing guidance.

3) Pricing model
   - Set price formula by material and print time.
   - Add minimum order value and rush fee policy.

Deliverables
- files/umbalabs3d_marketplace_listings.md

Phase 3 - Week 2-3 (B2B Outreach)
1) Outreach kit
   - Create one-page capability sheet and scripts.
   - Prepare 3 case study placeholders.

2) Target list
   - Build a list of 30 contacts: architects, product designers, event planners.

3) Pilot offers
   - Offer a pilot discount or sample to early B2B partners.

Deliverables
- files/umbalabs3d_outreach_kit.html
- files/umbalabs3d_outreach_kit.md

Phase 4 - Ongoing (Optimization)
- Update weekly capacity on the website.
- Replace placeholder testimonials with real quotes.
- Track conversion rate, average order value, and reprint rate.

Key Metrics
- Quotes per week
- Quote-to-order conversion
- Average order value
- Reprint rate
- On-time delivery rate

Dependencies
- Real WhatsApp number
- Real photos of printed work
- Actual weekly capacity number
