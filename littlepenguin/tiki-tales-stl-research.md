**Key Insight**: The Tonie ecosystem uses RFID/NFC chips + magnets in the feet. For Tiki Tales, you could:
1. Use the RFID base designs as inspiration for your own NFC-integrated figurine base
2. Design a universal base that snaps onto/into the bottom of any figurine
3. The base would house the NFC tag that identifies which character/story to play

## LICENSING SUMMARY

### MakerWorld Commercial Terms
- Free for personal use
- **Commercial use requires "Commercial License Membership"** ($3-$300/month set by individual creators)
- Creators need 200+ followers and 600+ prints to offer commercial memberships
- 10% platform fee + PayPal fees

### Printables License Notes
- Each model has its own license set by the creator
- Many use CC-BY-SA (commercial OK) but you must check each model individually
- Some creators (like Matthias L) offer commercial licenses via Patreon tiers
- Printables pages returned 403 errors when fetching directly -- must check licenses manually on site

### Thingiverse License Notes
- Each model has a creator-selected CC license
- **CC-BY 4.0** and **CC-BY-SA 4.0** both allow commercial use
- CC-BY-NC variants do NOT allow commercial use
- Always verify license on the actual thing page before using

## RECOMMENDATIONS

### Immediate Actions (Use existing CC-BY/CC-BY-SA models)
1. Download and test-print the Tier 1 models for: Lion, Elephant, Turtle, Hare, Snake
2. Scale each to 60-70mm tall and evaluate printability without supports
3. Round any sharp low-poly edges in a mesh editor (Blender/Meshmixer) for kid-safety

### Custom Design Needed For:
1. **Anansi (Spider)** -- No good commercially-licensed option exists. Design a cute, rounded spider with chunky legs.
2. **Jogoo (Rooster)** -- No commercially-licensed option found. Design custom or negotiate with existing designers.
3. **Boy/Girl human figures** -- Design simple peg-doll style figures. Unique IP is better anyway.
4. **NFC base integration** -- Design a universal snap-on base housing the NFC tag, inspired by the Tonie base designs above.

### Consider Designing All Custom
Given that 3/8 characters need custom design anyway, it may be worth designing the entire set from scratch for:
- Consistent visual style across all figurines
- Full IP ownership (no attribution requirements)
- Optimized specifically for 60-80mm, no-support FDM printing
- Kid-safe design from the ground up
- Unique brand identity for Tiki Tales

The open-source models above can serve as excellent **reference and inspiration** even if you go fully custom.
