## 1. Eclipse AVN-R8W (OEM Toyota Wide Format)

### Overall Chassis Dimensions (from Operation Manual p.204)

- **Standard models (AVN-S8, AVN-R8, AVN-D8):**
  - Width: 180mm
  - Height: 100mm
  - Depth: 165mm
- **Wide models (AVN-S8W, AVN-R8W, AVN-D8W):**
  - Width: 205.5mm
  - Height: 104mm
  - Depth: 165mm

### Weight

- Standard models (S8/R8/D8): approx. 2.4kg
- Wide models (S8W/R8W/D8W): approx. 2.5kg

### Common Specs (all models)

- Power: DC13.2V (10.5-15.8V), ground-only
- Operating temp: -20 to +65 deg C
- Storage temp: -40 to +85 deg C
- Current: 0.5W x4 output approx. 2.5A / Max approx. 12A
- Rated output: 16W x4 (1kHz, 1% THD, 4ohm)
- Max output: MOS-FET 49W x4
- Speaker impedance: 4ohm per channel
- Display: 7-inch wide low-reflection panel, 800x480x3(RGB), TFT active matrix, LED backlight

### GPS Antenna

- Dimensions: 53.3(W) x 14(H) x 8.5(D) mm (excl. protrusions)
- Weight: approx. 68g

### Communication Unit (AVN-S8/S8W only)

- Dimensions: approx. 50 x 70 x 16mm
- Weight: approx. 43g

### Installation Manual (Toritsuke Setsumeisho)

**Source:** DENSO TEN official — separate 8-page documents for standard and wide models

**Wide model (AVN-S8W/R8W/D8W) key installation details:**

- Mounting brackets: Toyota/Daihatsu Type A and Subaru Type B bracket hole positions
- Bracket screws: M5x8 hex bolts
- Antenna: Film antenna installation on windshield
- Wiring: 20-pin connector, plus TV antenna, GPS, communication unit, microphone, steering switch connections

## 2. Sony XAV-AX8500

### Overall Dimensions (from Help Guide Specifications)

- **Maximum (overall):** approx. 269 x 167 x 263mm (W x H x D)
  - This includes the floating 10.1" touchscreen
- **Mounting (chassis):** approx. 178 x 50 x 158mm (W x H x D)
  - This is the single-DIN chassis body
- **Weight:** approx. 2.6kg

### Chassis Dimensions (from Breakers Stereo product page, in inches)

- Width: 7.075" (179.7mm)
- Height: 1.985" (50.4mm)
- Depth: 6.655" (169.0mm)

NOTE: Slight discrepancy in depth between Help Guide (158mm) and retailer listing (169mm/6.655"). The Help Guide figure (158mm) is likely the chassis-only depth; the retailer figure (169mm) may include the rear connector protrusion.

### Connector Bracket Adjustment

- **Width adjustment:** 25mm total range, 5 steps, 12.5mm pitch
- **Depth adjustment:** 20mm total range, 3 steps, 10mm pitch
- **Torque:** 1.5-2.0 N-m

### Display

- 25.7cm / 10.1" wide LCD color monitor
- Resolution: 1280 x 720 (2,764,800 pixels with RGB subpixels)

### Mounting Hardware (from Service Manual parts list)

- Mounting screws: 5 x max 9mm (x4)
- Joint cover fixing screws: P-HEAD M3.0 x 8.0mm BLACK (x1)
- Joint part fixing screws: P-HEAD M4.0 x 6.0mm SILVER (x4)
- Screw assembly: T-HEAD M5.0 x 9.0mm SILVER (x4)

### Power

- 12V DC, negative ground
- Rated current: 10A
- Audio: 20W RMS x4 (4ohm, <=1% THD+N) / 55W peak x4

## 3. Key Dimensions for Trim Design

### The critical fit dimensions:


| Measurement        | Eclipse AVN-R8W (OEM) | Sony XAV-AX8500 (New) | Delta                      |
| ------------------ | --------------------- | --------------------- | -------------------------- |
| Chassis Width      | 205.5mm               | 178mm                 | -27.5mm (13.75mm per side) |
| Chassis Height     | 104mm                 | 50mm                  | -54mm                      |
| Chassis Depth      | 165mm                 | 158mm                 | -7mm                       |
| Face/Screen Width  | ~205.5mm (flush)      | 269mm (floating)      | +63.5mm overhang           |
| Face/Screen Height | ~104mm (flush)        | 167mm (floating)      | +63mm overhang             |


### Toyota Prado 150 dash opening: ~200 x 100mm (confirmed by client photos)

### Trim adapter requirements:

- Outer: 200 x 100mm (match Prado dash opening)
- Inner: 178 x 50mm (DIN cutout for Sony chassis)
- Sony DIN cage sits high in the opening (biased toward top)
- Sony floating screen extends beyond trim in all directions
- Gap fill needed: primarily below and on sides of DIN cage
